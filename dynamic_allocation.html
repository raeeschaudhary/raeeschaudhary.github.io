Dynamic Allocation and Analytics

This project streamlines the dynamic allocation problem with an interactive application to support salespeople in efficiently managing their allocation requests. This greatly reduces their time to manage daily allocation checks. It automates the allocation checks and provides a one-stop solution for the allocation problem, saving thousands of dollars and work hours.
